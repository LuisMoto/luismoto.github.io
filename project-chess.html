Modular Chess Training Suite

A Desktop Application for Advanced Chess Study

PYTHON 3.12 OOP MVC PATTERN STOCKFISH ENGINE

Technical Architecture

Unlike monolithic applications, this suite was designed using an Object-Oriented (OOP) approach with a focus on modularity and resource optimization. Each tool operates as a self-contained module, orchestrated by a central kernel.

Dynamic Orchestration

Implemented Lazy Loading in the main kernel (app_principal.py). Modules are only imported into memory when invoked, ensuring a minimal RAM footprint during idle states.

State Management

A custom Game State Manager (logica_ajedrez.py) handles complex chess rules including FEN parsing, castling rights, and pseudo-legal move generation with zero external dependencies.

UI/UX Consistency

Utilized a centralized Style Dictionary (style.py) to enforce a dark-mode professional aesthetic across all 13+ modules, adhering to the DRY (Don't Repeat Yourself) principle.

1. Stockfish Engine Integration

For the Checkmate Trainer, I integrated the Stockfish C++ binary. The app communicates with the engine via pipes to provide real-time evaluation and dynamic opponent responses, allowing users to practice endgame techniques against a Grandmaster-level opponent.

  • Engine: Stockfish 16.1

2. Applied Mathematics: Elo Analytics

Developed a professional statistical tool (calculadora_elo.py) using CustomTkinter. This module implements official chess rating formulas to calculate tournament performance ratings (Rp) and expected scores based on opponent strength.

By leveraging mathematical models of probability, the app provides players with data-driven insights into their competitive progress and rating volatility.

  • Model: Elo Rating System

3. Data Persistence & CRUD

The Flashcard System implements a full CRUD lifecycle. It utilizes JSON pipelines to store personalized positional puzzles, allowing users to build and save their own opening repertoires and tactical patterns locally.
